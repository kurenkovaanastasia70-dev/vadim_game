Растровые изображения: пиксели, палитра, битовая глубина и объём

Цель занятия: научиться решать задачи на объём растрового изображения и максимально возможное количество цветов в палитре.

Главная идея

Пиксель

Изображение как таблица

Растровое изображение состоит из маленьких точек — пикселей. Размер 256 x 256 означает 256 столбцов и 256 строк.

Палитра

Цвет как номер

В палитровом изображении каждый пиксель хранит не сам цвет, а номер цвета в палитре. Чем больше цветов, тем больше бит нужно на пиксель.

Биты

Глубина цвета

Если на пиксель отводится i бит, то можно закодировать 2^i разных цветов.

Объём

Формула

Объём без сжатия равен количеству пикселей, умноженному на количество бит на пиксель.

Формулы

N = ширина x высота

N — количество пикселей.

I = N x i

I — информационный объём в битах, i — битовая глубина, то есть количество бит на один пиксель.

K = 2^i

K — максимально возможное количество цветов в палитре.

1 Кбайт = 1024 байта, 1 байт = 8 бит

В задачах ЕГЭ/ОГЭ обычно Кбайт считают через 1024 байта, если не сказано иначе.

Алгоритм решения

  1. Найти количество пикселей: ширина x высота.
  2. Перевести объём файла в биты: Кбайт x 1024 x 8.
  3. Найти бит на пиксель: общий объём в битах / количество пикселей.
  4. Найти количество цветов: 2 в степени «бит на пиксель».
Дано: 256 x 256 пикселей, 80 Кбайт.
Пикселей: 256 x 256 = 65536.
Бит всего: 80 x 1024 x 8 = 655360.
Бит на пиксель: 655360 / 65536 = 10.
Цветов: 2^10 = 1024.

Задача 1

Имеется растровое изображение размером 256 x 256 пикселей и объёмом 80 Кбайт. Определите максимально возможное количество цветов в палитре изображения.

Показать решение 
1. Количество пикселей:
256 x 256 = 65536.

2. Переводим объём в биты:
80 Кбайт = 80 x 1024 байт = 81920 байт.
81920 x 8 = 655360 бит.

3. Бит на пиксель:
655360 / 65536 = 10 бит.

4. Цветов:
2^10 = 1024.

Ответ: 1024 цвета.

Задача 2

Растровое изображение размером 128 x 128 пикселей занимает 12 Кбайт. Сколько цветов максимально может быть в палитре?

Показать решение 
1. Пикселей:
128 x 128 = 16384.

2. Объём в битах:
12 x 1024 x 8 = 98304 бит.

3. Бит на пиксель:
98304 / 16384 = 6 бит.

4. Цветов:
2^6 = 64.

Ответ: 64 цвета.

Задача 3

Изображение размером 512 x 256 пикселей использует палитру из 256 цветов. Найдите объём изображения в Кбайтах без сжатия.

Показать решение 
1. Цветов 256, значит:
256 = 2^8, поэтому i = 8 бит на пиксель.

2. Пикселей:
512 x 256 = 131072.

3. Объём в битах:
131072 x 8 = 1048576 бит.

4. Переводим в Кбайты:
1048576 / 8 = 131072 байт.
131072 / 1024 = 128 Кбайт.

Ответ: 128 Кбайт.

Задача 4

Растровое изображение размером 320 x 240 пикселей имеет объём 75 Кбайт. Определите битовую глубину и максимальное количество цветов.

Показать решение 
1. Пикселей:
320 x 240 = 76800.

2. Объём в битах:
75 x 1024 x 8 = 614400 бит.

3. Бит на пиксель:
614400 / 76800 = 8 бит.

4. Цветов:
2^8 = 256.

Ответ: 8 бит на пиксель, 256 цветов.

Типичные ошибки

Забыли умножить на 8

Кбайты переводятся в байты, а затем байты в биты. Если забыть x8, битовая глубина получится в 8 раз меньше.

Перепутали цвета и биты

Если палитра 256 цветов, это не 256 бит на пиксель, а 8 бит, потому что 256 = 2^8.

Потеряли 1024

В учебных задачах 1 Кбайт обычно равен 1024 байтам, а не 1000.

Сжатие

Если в задаче не сказано про сжатие, считаем изображение без сжатия и без служебных заголовков.
