Урок: API и Google Sheets для игры

60 минут, 7 класс, с практикой в проекте

0-5 мин

Зачем игре API

API позволяет менять контент (задачи, ачивки, тексты) без изменений кода игры и без новой сборки.

Без API

Каждая правка = редактирование файлов проекта и новый релиз.

С API

Таблица обновляется, игра при старте подтягивает свежие данные.

5-12 мин

Основные термины

  • Client: игра, которая запрашивает данные.
  • Server: сервис Google, который отдаёт ответ.
  • Endpoint: URL точки доступа.
  • Request: запрос (обычно GET).
  • Response: ответ (CSV/JSON).
  • Status code: 200, 404, 500.
12-18 мин

Путь данных по шагам

  1. Игра читает URL таблицы из переменной окружения.
  2. Отправляет HTTP GET.
  3. Получает CSV-строки.
  4. Парсит колонки в объекты ачивок.
  5. Валидирует числа: target, reward.
  6. Передаёт данные в менеджер прогресса.
18-28 мин

Подготовка таблицы

Колонки должны быть строго такими:

id,title,description,event_key,target,reward

Export URL:

https://docs.google.com/spreadsheets/d/<ID>/export?format=csv&gid=<GID>
28-36 мин

Почему нужен fallback

Риск

Нет сети, ошибка Google, таблица пустая.

Решение

Если онлайн-таблица не загрузилась, игра использует локальный CSV.

36-43 мин

Безопасность API

  • Для урока: только публичный read-only CSV.
  • Не хранить приватные ключи в репозитории.
  • Для приватных данных нужен backend-сервер.
43-50 мин

Практика ученика

  1. Добавить новую ачивку в Google Sheet.
  2. Запустить игру с переменной окружения.
  3. Проверить лог: подтянулась таблица достижений: ...
$env:GOOGLE_SHEETS_ACHIEVEMENTS_CSV_URL="https://docs.google.com/spreadsheets/d/<ID>/export?format=csv&gid=<GID>"
python main_work.py
50-56 мин

Частые ошибки и диагностика

  • Неверный gid листа.
  • Лист не опубликован в интернет.
  • Нечисловые значения в target/reward.
56-59 мин

Короткий квиз

Почему игра должна иметь fallback на локальный CSV?

Ответ: чтобы работать даже при проблемах с интернетом/таблицей.

59-60 мин

Итог урока

  • Поняли API и схему client-server.
  • Сделали Google Sheet как внешний источник данных.
  • Научились подключать CSV в игру через URL.
